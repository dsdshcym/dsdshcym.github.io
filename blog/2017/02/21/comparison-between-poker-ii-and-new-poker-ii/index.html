Comparison between Poker II and New Poker II

起因

寒假开始老的 Poker II 在服役了两年多之后终于撑不住了，连接上电脑时 Capslock 灯闪烁不停，但是按任何键都没反应。

起初还能通过反复插拔来碰碰运气，掌握好角度能让键盘连接成功。后来越来越难，于是就考虑换个键盘。

抉择

刚好情人节要到了，女朋友就说她来出资，预算 1000 左右，买一个好一点的键盘（不会两年多就坏的那种），彻底解决这一问题。

我因为习惯了 Poker II，对 60% 左右的小键盘情有独钟，无形中增大了很多挑键盘的难度。

WASD

刚开始看上了 WASD VP3，因为可以自定义很多东西，比如轴体（红轴）、颜色（黑）、印刷（无刻 + Linux 小企鹅）等等。

但是购买比较麻烦，还要美国代购，风险比较高，等待时间比较久，就犹豫了好几天没有下手没有出手。

HHKB

跟女友提了一句「一般都是买 HHKB，一步到位」，然后女友就很意外地说直接上 HHKB，以后就不会有键盘方面的需求了。

虽然我也很想尝试一下这个神器，但是接近 2000 的价格摆在那里，还是有点舍不得花这么大的价钱来买一个历史有点悠久（接口还是 Mini USB）的键盘（HHKB Pro 2 2006 年， HHKB Pro 2 Type-S 2011 年）。

再把目光瞄向 HHKB Pro BT，这款 2016 年新出的键盘价格更是直逼 2000，想想那老旧的 Micro USB 和 Bluetooth V2.0，就不是很想出手。

另外，HHKB 的键位也比 Poker II 少一些，让想用 Ctrl Key 作为 Hyper Key 的我有点尴尬。

Ctrl Key 在 Capslock 键上本来是个加分项，但是我已经在系统层面把 Capslock 设置成了 Ctrl。这一特性反而变成了减分项（因为要考虑和笔记本内置键盘的区别对待）。

最后，尝试过同事的 HHKB，感觉和机械键盘相比差别也不是特别大，暂时就打消了买入 HHKB 的念头。

Minila Air

考虑到蓝牙机械不得不想到的键盘。但是

  • 键位太过特别
    • Z 键和普通键盘有偏移
    • 为了方向键把右 Shfit 切分开，对不需要方向键、依赖右 Shift 的我来说就直接不考虑了。

New Poker II

大约是在搜索 60% 键盘的时候搜到了这么一篇什么值得买上的文章 它之所以是它：全新 IKBC Poker 2 机械键盘评测报告。_值友评测_什么值得买。我才发现 2017 年一月，ikbc 又出了复刻版的 Poker II（下称 New Poker II），大约是 Poker 3 评价太差，就又把经典 Poker II 拿出来升级一下。

本以为是换汤不换药的升级，没想到还很良心：

  • USB Type C 接口，并且附赠 USB Type A-Type C 和 USB Type C-C 的线，这对已经有 Type C 设备（OnePlus 2）以及想换有 Type C 接口的笔记本（Thinkpad X1 Carbon 2017）的我来说吸引力很大。
  • 接头从直出变成了 90 度弯头，并且提供了出线槽，让线可以从键盘底部左右两侧出来，不用怕顶到前面的笔记本等其他东西。（相信这样的设计能很大程度减少我那老 Poker II 接触不良的惨剧）
  • 支持三层 Layer 的编程，虽然我都用不到，但聊胜于无
  • 支持直接改变成 Dvorak 键位，虽然我也不会尝试了，但是又向 HHKB 迈出了一步呢（笑

于是就果断让女友在京东下单了一把黑色红轴，第二天就到了（京东快递真的快）。

比较

拿到键盘以后往电脑上一插，完全不需要任何改动就可以像 Poker II 一样开始工作了，但是在体验方面又是完全不同的感受。

New Poker II vs Poker II

  1. 键帽

    这次的键帽还是 PBT 材质。虽说 PBT 不容易打油，但是用了两年的 Poker II 还是有点打滑了。

    于是这次的键帽还加上了磨砂的手感，第一次摸上去简直惊艳。比 Poker II 的光滑键帽高到不知道哪里去了。

    但是现在用了一个多星期，磨砂的手感没刚开始前那么明显了，希望至少能稳定在现在这个水平吧。

  2. 反馈

    不知道是不是新键盘的原因，虽然同是红轴，但是在 New Poker II 上的反馈感就比 Poker II 上要强上好多，每次按下去都有稳定的反馈。让打字体验好了不少。

    没有比较不知道，一比较发现 Poker II 的蛮多按键都不是很稳定。特别是用了卫星轴的那几个键（空格、退格、回车等等，不太确定，待考），和普通的干脆按键差别很大，肉肉的，能感觉明显的不同。New Poker II 上就统一了很多，至少除了空格以外的几个键都能和普通键差不多统一了（和普通键变肉也有关）。

    红轴还是和以往一样直上直下，但是更稳定也使得打字更快速了。

  3. 声音

    和反馈一样，New Poker II 的声音也变了，变得更沉闷，没那么清脆了。这一点说不出是好是坏，但是对我来说没有太大的影响。倒是从噪音角度来说要比 Poker II 小了不少。可以算是优点吧。

  4. 接口

    USB Type C 太厉害了。OnePlus 2 和 New Poker II 的数据线都可以共用，就是不知道能不能用 USB Type C-C 接口连接两个，不过除了装 X 也没有什么用处，我也就不尝试了。

    送的 Type C-C 线也让我对购买 Type C 接口的笔记本更有信心，毕竟可以少一个转接口了。

New Poker II vs HHKB

又用了同学的 HHKB 打了一个小时左右的字，可以说还是有差距的（毕竟一个 500 价位，一个 2000 价位）。

特别是键盘的做工，看起来差距比较大。New Poker II 键帽下面有明显的瑕疵。但是在打起字来的时候注意不到，也就不算特别大的缺陷吧。

HHKB 的反馈感也更好，虽然红轴已经很优秀了，但是和静电容一比还是感觉差了一点。不过我个人还是不太喜欢静电容那肉肉的感觉，跟果冻似的，不太干脆。

总结

我非常亲幸在一月 New Poker II 刚推出之后才换键盘，如果我的 Poker II 早一点挂掉，我就很可能会错过这么一款优秀的拥抱未来的键盘了。Poker II 见证了我从一个还没入门的程序员到一个 Junior+ Rails Engineer 的旅途，我很高兴它的复刻版陪伴我走接下来更远的路。
