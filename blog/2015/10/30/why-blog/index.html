Why Blog?

序

其实我之前在 Github 上也有一个用 Hexo 的搭建的博客， 2014 年末开始写的，大约也有 几篇博文。但是最后还是没有坚持下来，我想大约是动力不够，不足以让我克服不能在编辑 器里一站式地完成博客写作、发布的不爽。

而这次， Emacs 让我能在 Emacs 里轻松地完成这件事情，这个博客应该也能坚持一段日子 吧。

当然，还有一些其他的原因，也就一起写在这篇博客里当个开头好了。

分享

让我再次冒出写博客的念头，还是要归功于 Emacs 和 Org-Mode 。

某天，我在折腾 Emacs 时发现了 org-mac-link 这个 package ，配合 Org-mode 的 Capture 可以方便地一键抓取 Safari 的当前页面。我觉得我的工作流因为这个小小的改进， 提高了不止一点。当时就想把这个分享给别人，但苦于没有渠道。

当然这个念头已经产生了好久了，毕竟用了 Emacs 这种神器，便总是有一种想把自己的 Tips 分享出去的冲动。（上学期本有机会在班级里做个分享，但因为阿卡林属性而作罢了）

学习

离开始学习 Vim 过了快两年了，转用 Emacs 也快一年了，期间一直在用 RSS 跟进 Vimcasts, Planet Emacsen 这些网站。看多了 Sacha, Chen Bin 他们的文章，自然也想向 他们学习，把自己的经验写出来。如果能吸引到一些读者，互相交流，从中学习，就再好不 过了。

锻炼

自己日常不怎么说话，也希望借这个博客，锻炼一下自己的表达能力，防止自己的语言能力 退化为零。
